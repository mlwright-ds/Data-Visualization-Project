📊 Data Visualization Project in R
📌 Overview

This project demonstrates a variety of data visualization techniques in R using libraries such as ggplot2 and the tidyverse.

The goal is to explore how visual design choices—such as color, transparency, and layout—impact interpretability, accessibility, and storytelling in data.

🛠️ Technologies Used
R
ggplot2
tidyverse
reshape2
qcc
📂 Project Structure

The project consists of multiple visualization examples, each focusing on a specific concept:

📈 Visualizations Included
🔹 1. Overlayed Histograms (Gender Comparison)
Simulated dataset comparing male and female weights
Uses transparency (alpha) to overlay distributions
Demonstrates:
Distribution comparison
Visual contrast techniques

📌 Key Insight: Transparency helps reveal overlapping distributions without losing clarity.

🔹 2. Histogram with Density Plot
Combines histogram and density curve
Improves understanding of data distribution shape

📌 Key Insight: Density overlays provide smoother insight into distribution trends.

🔹 3. Accessibility-Focused Pie Chart
Displays survey responses:
Bored (12%)
Not Great (6%)
OK (14%)
Kind of Interested (30%)
Excited (36%)
Uses color choices and labels for accessibility

📌 Key Insight: Proper labeling and color selection improve readability for color-blind users.

🔹 4. Bar Chart (Aesthetic Comparison)
Simple stacked bar chart
Demonstrates layout and visual clarity differences
🔹 5. Histogram (Clean Design)
Clean histogram with:
Black borders
White fill
Controlled bin width

📌 Key Insight: Minimalist design improves readability.

🔹 6. Correlation Heatmap
Uses numeric dataset correlations
Displays relationships between variables
Includes:
Color gradient (white → red)
Annotated correlation values

📌 Interpretation Guide:

0.01–0.30 → Low correlation
0.30–0.65 → Moderate correlation
0.65–1.00 → High correlation
🔹 7. Pareto Chart
Visualizes frequency of categorical data
Includes cumulative percentages

📌 Use Case:

Quality control
Identifying most impactful categories (80/20 rule)
🔹 8. Annotated Scatter Plot (Magic Quadrant)
Plots companies based on two dimensions
Divides space into quadrants:
Leaders
Visionaries
Challengers
Niche Players

📌 Key Insight: Strategic positioning visualization for competitive analysis.

🎯 Key Concepts Demonstrated
Data distribution visualization
Visual hierarchy and focus
Accessibility in charts
Correlation analysis
Business-oriented visualizations
Annotation and storytelling
